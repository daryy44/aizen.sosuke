Задание практики

Тема: Решение квадратного уравнения с использованием языка программирования Python.

Цель: Научиться применять условные конструкции, работать с математическими формулами, а также вводом и выводом данных в Python.

Условие задачи:

Написать программу, которая решает квадратное уравнение вида ax² + bx + c = 0, где a, b и c — коэффициенты, вводимые пользователем.

Пример кода на Python:


import math

print("Решение квадратного уравнения ax² + bx + c = 0")
a = float(input("Введите коэффициент a: "))
b = float(input("Введите коэффициент b: "))
c = float(input("Введите коэффициент c: "))

discriminant = b**2 - 4*a*c

if discriminant > 0:
    x1 = (-b + math.sqrt(discriminant)) / (2*a)
    x2 = (-b - math.sqrt(discriminant)) / (2*a)
    print(f"Два корня: x₁ = {x1}, x₂ = {x2}")
elif discriminant == 0:
    x = -b / (2*a)
    print(f"Один корень: x = {x}")
else:
    print("Действительных корней нет.")

Ожидаемый результат:

  • Пользователь вводит значения коэффициентов a, b и c.
  • Программа рассчитывает дискриминант.
  • Выводит корни уравнения в зависимости от значения дискриминанта.

Это базовая задача помогает закрепить навыки программирования, работы с вводом/выводом и математикой в Python.
